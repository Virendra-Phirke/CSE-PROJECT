🧪 MCQ Portal - Shared Test Link Debug Tool

This tool helps debug shared test link access issues. Follow these steps:

📋 Steps to Test Shared Test Links

  1. Login as Teacher: Go to localhost:5173 and login as a teacher
  2. Create a Test: Create a new test/quiz from the teacher dashboard
  3. Generate Share Link: Use the "Share" button to generate a test link
  4. Test as Student: Open the share link in an incognito window or different browser
  5. Login as Student: Login with a different account (student role)
  6. Take the Test: Try to take the test and submit results

🔍 Common Issues & Solutions

Issue 1: "Test not found"

  • Check if the test ID in the URL is correct
  • Verify the test exists in the database
  • Check if the test is marked as active

Issue 2: "User profile error"

  • Student profile not created in database
  • UUID generation or caching issues
  • Database foreign key constraints

Issue 3: "Cannot submit test results"

  • Database permissions (RLS disabled?)
  • Foreign key constraint violations
  • Student ID UUID issues

Issue 4: "Authentication errors"

  • Clerk configuration issues
  • Missing environment variables
  • Role setup problems
